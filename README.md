# 🗂️ Ransomware Tabletop Exercise Kit

*By S. Naz · Cybersecurity, Cloud & AI Security Leader*

A ready-to-run tabletop exercise kit for testing ransomware response — from detection to executive comms — before the real thing happens.

![Tabletop Exercise](https://img.shields.io/badge/Tabletop_Exercise-1F4E79?style=flat-square) ![Ransomware](https://img.shields.io/badge/Ransomware-D22730?style=flat-square) ![Incident Response](https://img.shields.io/badge/Incident_Response-00A36C?style=flat-square) ![Facilitation](https://img.shields.io/badge/Facilitation-0052CC?style=flat-square)

## What's Inside

- A full scenario script with timed injects, from initial detection through executive decision points
- Facilitator guide covering how to pace the exercise and what to probe for at each stage
- Role cards for IT, legal, communications, and executive participants
- After-action report template that turns exercise gaps into tracked remediation items

## Key Practices

- The scenario forces a ransom-payment decision point — the part most tabletop kits skip
- Exercise is designed to surface communication gaps, not just technical response gaps

## Status

- Actively maintained — adding a cloud-ransomware variant scenario.
